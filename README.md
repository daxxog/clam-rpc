# ClamRpc

  A command line RPC client for CLAMs.

  [![Build Status][travis-image]][travis-url]

Install
-------
stable
```bash
npm install -g clam-rpc
```
edge
```bash
npm install -g https://github.com/daxxog/clam-rpc/tarball/master
```
